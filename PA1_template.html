To fill the missing Dataset below strategy has been applied:-

1.Extracting missing values with NA and assigne them to dataset data. 2.Matching the data for missing values with 5 minutes interval values from dataset data and replacing them with the Average values calculated under dataset avg_agg in previous problem.The datset data will then have values of NA replaced by Avg values from avg_agg. 3.Removing NA values from original dataset file. 4.Merging the dataset data and dataset file after removing NA for dataset file. 5.The new dataset avg_data_new with missing value replaced by Avg value at 5 minutes interval.
